About Project

Web App is dedicated to status of Corona Virus spread in India. It fetches data from an API located at https://coronastatusindia.herokuapp.com/ which fetches data by scraping website of Ministry of Health and Family Welfare (https://www.mohfw.gov.in)

This project is for educational purpose as well as to show stats of corona virus in India.
